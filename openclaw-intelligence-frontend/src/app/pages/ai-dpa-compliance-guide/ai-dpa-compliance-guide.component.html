Key Takeaways

1. DPAs are mandatory. As a business processing client data through AI APIs, you need a valid DPA (Art. 28 GDPR) with every provider whose API you use to process personal data.

2. Most are auto-incorporated. Anthropic, Google, Microsoft, AWS, and Mistral require no separate signing. OpenAI requires an online form. Cohere requires email. Hugging Face requires Enterprise plan.

3. Free tiers lack coverage. Always use business/paid API tiers for client work involving personal data.

4. EU data sovereignty. Mistral AI (France) offers the strongest EU-native option. Azure provides EU Data Boundary. AWS and Google offer region selection.

5. Azure OpenAI ‚â† OpenAI. Azure OpenAI data stays within Microsoft and is never shared with OpenAI ‚Äî a crucial compliance distinction.

6. Document everything. Maintain a register of provider DPAs per client engagement (Art. 30 GDPR). Subscribe to subprocessor notifications for each provider you use.

7. Review regularly. DPAs evolve. Set calendar reminders to check for updates, especially after major regulatory changes.
